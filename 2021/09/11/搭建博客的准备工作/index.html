搭建博客的准备工作

Posted on 2021-09-11 | In 搭建博客 |

使用hexo和 GitHub搭建一个博客

做这个博客的时候也遇到很多的问题。经过个人的总结之后得到了以下的思路

  • 安装git node,js 和hexo框架
  • 部署
  • 使用next主题 对博客修正
  • 博客各种功能的实现

准备的工具

Github Pages Node.js Git Hexo Notepad++ typora

Github Pages

Github Pages是github提供的博客服务，只要创建一个特定格式的repository,Github Pages 会吧里面放置的信息生成一个网页

  1. 首先注册一个Github的账号 然后创建一个以 .github.io结尾的repository （ 例如Repository name：chenyuxl.github.io）勾选ADD a README file
  2. 这个时候打开 chenyuxl.github.io 网页 就可以看到 README file 里面的内容

Node.js

让JavaScript 运行在服务端的开发平台，它让JavaScript成为与PHP、Python、Perl、Ruby等服务端语言平起平坐的脚本语言。

  • 前往 https://nodejs.org/en/
  • 点击 8.9.1 LTS 下载
  • 安装
  • 打开 Command Prompt， 输入 node -v
  • 得到：v8.9.1 成功

Git

分布式版本控制系统

  • 前往 https://git-scm.com/
  • 点击 Downloads
  • 点击 Windows
  • 一般情况，下载会自动开始。如果没有，就点击 click here to download manually
  • 安装
  • 打开 Command Prompt， 输入 git --version
  • 得到：git version 2.15.0.windows.1

Hexo

Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Coding上，是搭建博客的首选框架。大家可以进入hexo官网进行详细查看，因为Hexo的创建者是台湾人，对中文的支持很友好，可以选择中文进行查看。

  • 打开 Command Prompt

  • 输入 npm install -g hexo-cli

  • 回车开始安装

  • 输入 hexo -v

  • 得到 hexo-cli: 1.0.4 等一串数据

    安装成功

Notepad++

方便的编辑软件，软件作者辱华，现在官网都不好找了，供选着使用

  • https://notepad-plus-plus.org/

typora

室友推荐的十分好用的makedwon编辑软件，后续会更新使用大全

  • https://www.typora.io/
